# DevRadar

Сервис-радар зависимостей: информация о пакетах и их экосистеме.

**Статус:** в разработке. Прото-фаза на plain Java (без фреймворков) — учебный маршрут от HTTP-клиента до собственного сервера.

## Структура

- `labs/` — одноразовые учебные эксперименты
- `docs/learning/` — конспекты по квестам
